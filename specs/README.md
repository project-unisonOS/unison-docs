# Unison Specs Index

This directory hosts the specifications and schemas formerly in `unison-spec`. Key entries:

- `SETUP.md` — how to work with the specs
- `MAINTAINERS.md` — contacts/roles
- `CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`, `SECURITY.md` — policies
- `constraints.txt` — pinned dependency constraints for reproducible tooling
- `docs/` — architecture, safety, accessibility, compatibility
- `specs/` — protocol/spec documents (e.g., unified messaging, event envelope)
- `schemas/` — JSON/YAML schemas (e.g., event envelope)
